## Set Layer Editable

Sets a layer to be editable in the uncommitted engine.

### Syntax

```js
Actions.setLayerEditable(id, editable)
```

### Parameters

`id: number`

The id of the layer to make editable.

`editable: boolean`

Whether or not the layer should be editable.
