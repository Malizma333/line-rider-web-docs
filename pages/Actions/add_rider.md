## Add Rider

Adds a new rider to the uncommitted track.

### Syntax

```js
Actions.addRider(rider)
```

### Parameters

`rider: Rider`

A [`Rider`]({{ site.baseurl }}{% link pages/External/rider.md %}) object to add.
