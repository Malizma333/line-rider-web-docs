## Set Layer Visible

Sets a layer to be visible in the uncommitted engine.

### Syntax

```js
Actions.setLayerVisible(id, visible)
```

### Parameters

`id: number`

The id of the layer to make visible.

`visible: boolean`

Whether or not the layer should be visible.
